ARS

The Automatic Registration Service (ARS) feature provides automatic data application registration for walkie-talkies. After opening the walkie-talkie, the intercom is automatically registered to the server. This function is used with the data application, that is, any data transfer on this channel is associated with the application server.

Option
DisableDisable ARS function.
System changeEnable ARS for a single site.

Note
  • Only digital mode supports this feature.
