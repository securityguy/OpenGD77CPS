Tx Interruptible Frequencies

If the frequency support can interrupt voice transmission, you need to enable this feature. For support for direct mode (off-line mode) can interrupt the transmission frequency, the air slot assignment is undefined. So this is based on frequency rather than channel (time slot) considerations. Enabling this parameter in some cases will slightly increase the channel access time but reduce the frequency of the signal collision possibilities. This feature should not be enabled on this channel because the data reply channel does not support voice calls. This is a channel-wide function.

Note
  • Must disable the receive only function.
  • Only digital mode supports this feature.
