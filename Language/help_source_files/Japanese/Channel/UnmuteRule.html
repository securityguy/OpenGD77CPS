Unmute Rule

Sets the rules that determine when the speaker will sound the speaker to receive audio or data. This is a channel-wide function.

Option
Std Unmute, MuteSounds when the correct private line (PL) code is detected, mutes when missing PL code.
And Unmute, MuteSound is detected when the correct private line (PL) code and carrier squelch are detected and muted when missing PL code.
And Unmute, Or MuteSound is detected when the correct private line (PL) code and carrier squelch are detected, muted when the correct PL code is lost or the carrier is lost.

Note
  • The receive squelch type function can not be set to carrier squelch
