Timing Leader Prefernce

Set the time to wait for the intercom on the channel when the carrier signal with sufficient amplitude is detected on the analog scan list channel. This pause allows the intercom to have time to decode the analog system signaling data. If the decoded information is incorrect, the intercom will revert to the scan.

Option
PreferredThe radio is used as a channel time slot calibrator, such as a channel base station.
EligibelThe radio function as a time slot calibrator, but should follow the higher priority calibrator.
IneligibelRadio can not be used as a calibrator, such as frequent roaming Radio.

Note
  • This function is available when dual capacity direct mode (DCDM) is enabled.
  • Only digital mode supports this feature.
