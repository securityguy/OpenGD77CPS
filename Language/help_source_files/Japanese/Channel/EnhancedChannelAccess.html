Enhanced Channel Acsess

Allows the user to enable or disable enhanced channel access. Enabling this feature, when two or more interphones simultaneously initiate a call, the enhanced channel access function improves the reliability of the transmission by reducing airborne (OTA) collisions. You need to enable this feature on all walkie-talkies that access the channel. This is a channel-wide function.

Note
  • This function is disabled when the Dual Capacity Direct Mode (DCDM) function is enabled.
  • This feature is only available for repeater mode.
