PTTID Type

Set the decision to send walkie talkie press PTT when the PTTID type.

Option
NonePress or release PTT does not send or drop code
Only FrontPress PTT PTTID on the line code (the specific codeword in the signaling system DTMF PTTID online code can be set)
Only PostRelease PTT hair PTTID offline code (specific codewords in the signaling system DTMF PTTID offline code can be set)
Front & PostPress PTT PTTID on-line code; release PTT hair PTTID line code

Note
  • Only analog mode supports this feature.
