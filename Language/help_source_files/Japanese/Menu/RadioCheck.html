Radio Check

Allows the user to initiate a Radio Check request from the menu. Radio check allows a user/console operator to determine if a radio is active in a system without showing any indication to the radio's user.

Note
  • Disabling this feature disables the user from starting this feature from the radio's user interface. But this will not prohibit the walkie-talkie to respond to "radio check" command.
  • Only digital mode supports this feature.
