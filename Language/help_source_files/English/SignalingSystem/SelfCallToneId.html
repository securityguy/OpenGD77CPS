Self Call Tone/ID

When the intercom is connected to the PTT button, it is limited to send it.

Range
AlwaysThe radio continues to transmit the radio ID when selective calls are made.
FrontThe radio transmits the radio ID at the beginning of each selective call.

Note
  • This function is disabled when the selective call coding function is disabled (not selected).
  • This function is set to Always when the selective call hold time (in milliseconds) is set to 0 milliseconds.
  • Only digital mode supports this feature.
