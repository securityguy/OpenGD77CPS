Self Call Encode

This feature enables the radio to initiate selective calls. Alternative calls can reduce calls that are not of interest. Selective calls are especially used when the majority of transmissions are in the scheduler with a radio user or a group of walkie-talkie users while the other user does not care about the call.

Note
  • When this feature is disabled, the selective call hold time (ms) and the selective call tone function are disabled (unchecked).
  • Only analog mode supports this feature.
