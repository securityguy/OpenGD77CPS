PL Type

This feature indicates if Private Line (PL) decoding is required to unmute to a channel with activity during a scan operation. Disabling the need for CTCSS/DCS decoding increases the scanning speed.

Option
DisableNo PL decoding required.
Non-Priority ChannelPL decoding is required on non-priority scan list member channels.
Priority ChannelPL decoding is required only on priority list 1 or priority 2 scan list member channels.
Priority and Non-Priority ChannelPL decoding is required on all current scan list member channels.

Note
  • Before selecting the priority channel option, make sure there are priority members in the scan list.
  • Enabling this feature overrides the PL association of the scan list member channel.
