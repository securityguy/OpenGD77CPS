Text Message

The user can enter up to 144 characters at most. Valid characters are alphanumerics, spaces and special characters. The user can send the text message by assigning a short or long programmable button press (Text Message) or access the Text Messages feature via the Text Messages Menu feature.

Note
  • Only digital mode supports this feature.
  • Add, add to the minimum free text message number, the default content is empty.
  • Delete, delete the currently selected text message.
