Rx Only

The configuration channel is only received and does not use any of the transmit functions. All of the channel's transmit functions will also be disabled. Enabling this feature restricts the radio to receiving only unacknowledged data. This is a channel-wide function.

Note
  • The channel can not be set to any reply channel (e.g., an emergency reply channel) or a designated transmit channel (e.g., scan the specified transmit channel).
