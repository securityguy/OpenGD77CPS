Auto Scan

Allows the radio to automatically begin scanning when the user selects the current conventional channel. When disabled, the user is still able to invoke the scan operation, via a short or long programmable button press (Scan On/Off) or Scan (Scan Menu) feature. This is a channel-wide feature.

Note
  • If the Scan List function is set to None, this feature is disabled.
  • Double standby, scan does not work
