Voice Emphasis

By applying audio shaping filters to reduce the noise in the radio signal, improve the higher frequency of audio clarity. The type of filter used depends on the options selected. If no is selected, no filter is applied to the signal. The pre-emphasis is used to filter the transmit signal, which is used to filter the received signal. This is a channel-wide function.
