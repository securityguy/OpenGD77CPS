Tx Signaling System

During launch, all available DTMF signaling systems are associated for use.

Note
  • Before configuring, please configure the signaling system.
  • When the intercom is transmitted in the off-line mode, the reception settings including the receiving signaling system will be used for the transmission. However, the transmit signaling system is still required to be a valid DTMF system to enable the emergency settings for the specified DTMF system.
  • Must disable the receive only function.
