Receive

Enabling this feature causes an idle radio to automatically enter battery saver mode where it places certain radio functions on standby. After a certain duration or when there is any user button action, the radio returns to normal operation and checks the channel for incoming calls. If no calls are detected, it returns to the battery saver mode.

Note
  • Channel dual capacity direct mode function is turned on when power is not working
  • If the preamble is disabled (unchecked), this feature will be disabled.
