TX Preamble Duration

Preamble is a string of bits added in front of a data message or control message (Text Messaging, Location Messaging, Registration, Radio Check, Private Call, etcâ€¦) before transmission. This preamble prolongs the message in order to reduce the chances of the message being missed by the receiving radio. The Transmit (TX) Preamble Duration sets the duration of the preamble. This duration needs to be increased as the number of scan members increases on the target radio. This value can be increased in all the transmitting radios if scanning radios are often missing data messages. However, a larger preamble occupies the channel longer. Therefore, increasing the Transmit Preamble duration will increase the success rate of data received while other radios are scanning, but will decrease the amount of data that can be transmitted on the channel.

Range
Max8640
Min0
Step60

Note
  • If the duration is set to 0, the transmit preamble function is disabled.
  • If the radio configuration power is disabled, the function should be set to 0.
  • Only analog mode supports this feature.
